# Resume Hacker V2.0.0 Roadmap

This document outlines the major release plan for Resume Hacker.

## Vision

Transform "Resume Hacker" into a comprehensive job preparation suite that is free, private, and powerful.

## Key Features

- **Job Tracker**: Local-first Kanban board to track job application status.
- **Interview Prep**: AI-generated mock interview questions and behavioral answer assistance (STAR method).
- **LinkedIn Assistant**: Headline optimization and networking outreach templates.
- **Multi-Resume Management**: Save and switch between different resume versions for different roles.
- **Dashboard**: High-level overview of job search progress.

## Technical Architecture

- **Data Persistence**: LocalStorage for privacy and zero-backend costs.
- **UI Architecture**: Modular navigation shell with a sidebar.
- **AI Integration**: Continued support for Ollama (local) and cloud APIs.
